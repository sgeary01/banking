Quick commands

Reliable demo trigger (no code change):
kubectl scale deployment reporting-service -n banking --replicas=0 # fire ServiceDown kubectl scale deployment reporting-service -n banking --replicas=1 # clear
Inspect Elastic from the shell:
kubectl --context k3d-banking exec -n monitoring deploy/elasticsearch -- \ curl -s 'http://localhost:9200/banking-logs/_count' # 5 most recent structured logs kubectl --context k3d-banking exec -n monitoring deploy/elasticsearch -- \ curl -s 'http://localhost:9200/banking-logs/_search?size=5&pretty' \ -H 'Content-Type: application/json' \ -d '{"query":{"exists":{"field":"event"}},"sort":[{"@timestamp":"desc"}]}'
Cluster lifecycle:
kubectl config use-context k3d-banking kubectl get pods -A k3d cluster stop banking # pause to save resources k3d cluster start banking # resume make -C ~/gitrepos/resolve/banking k3d-rebuild # rebuild all + reimport + restart
